# EjercicioEED

##Ejercicio calculadora de números complejos

>###Este programa realiza estas operaciones con 2 números complejos:
>- Sumar
>- Restar
>- Multiplicar
>- Dividir
>- Obtener el resto

####El programa tiene:
1. Clase principal (main ProgramaComplejos)
2. Clase NumComplejos
3. Javadoc
